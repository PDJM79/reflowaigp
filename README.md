# GP Practice Management System

A comprehensive NHS GP practice management system for practices across Wales, England, and Scotland. Manages healthcare compliance, policies, HR, training, incidents, complaints, and regulatory requirements.

## Tech Stack

- **Frontend**: React, TypeScript, Vite, Shadcn UI, Tailwind CSS
- **Backend**: Express.js, Drizzle ORM
- **Database**: Neon Postgres

## Development

```sh
npm install
npm run dev
```

## Deployment
